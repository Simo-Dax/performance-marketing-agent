# /pm-seo-content — SEO Content Optimizer (SA7 / organico)

Esegui la skill **`directives/skills/41_seo_content_optimizer/SKILL.md`**.

Argomenti: $ARGUMENTS

## Cosa fare
1. Leggi e segui `directives/skills/41_seo_content_optimizer/SKILL.md`.
2. **Step 1 — Fetch:** recupera il contenuto dall'URL (`web_fetch`). Estrai title, slug, word count, heading structure, immagini, tipo di contenuto, profilo voce autore, topic core. Se fetch fallisce → chiedi il testo grezzo, non procedere con dati parziali.
3. **Step 2 — Keyword research:** 5-8 web search reali → mappa keyword sui 9 tipi (vedi `references/keyword-research-types.md`) con priority score Alto/Medio/Basso.
4. **Step 3-8:** ottimizza slug (`references/slug-formulas.md`), heading structure, meta description (2 varianti 150-160 char), alt text immagini, internal linking, integrazione keyword. **Preserva sempre la voce dell'autore** — niente keyword stuffing.
5. Lingua: default IT se il contenuto è in italiano (keyword research e suggerimenti nella lingua del contenuto).
6. Output report nell'ordine: Sintesi Analisi → Keyword Map → URL Slug → Heading Structure → Meta Description → Alt Text → Internal Linking → Integrazione Keyword → **Quick Win (top 5, ranked per impatto/effort)**.
