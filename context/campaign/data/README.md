# Campaign Data — dati first-party del cliente

Metti qui gli export di dati **propri del cliente** per `38_first_party_data_analysis` (`/pm-data-analysis`).

## Quantitativi
- GA4: traffico, funnel, coorti (CSV/export)
- Shopify: Sales over time, Customers over time, AOV, ordini, returning
- Google/Meta Ads: export spend/ROAS/keyword
- Gestionale / CRM

## Qualitativi
- Recensioni del brand (export)
- Ticket customer care / chat log
- Survey / NPS / interviste

> Se vuota, la pipeline procede con brief + ricerca pubblica. Con dati reali qui, SA3 usa baseline veri e gli insight sono più solidi. Nota: questa cartella contiene dati cliente — non versionare/pubblicare (vedi `.gitignore`).
