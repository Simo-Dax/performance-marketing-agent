# Vercel Web Design Guidelines — README

## Overview
Sistema di audit production-grade con 100+ regole su accessibilità, UX, performance e visual quality, ispirato agli standard Vercel.

## Quick Start
```bash
# Quando usare:
"Audit questa landing page prima del deploy"
"Verifica che la pagina rispetti gli standard accessibility"
"Check finale pre-produzione"

# Output: Report completo con score, critical issues, e fix suggeriti
```

## Categorie di Audit
1. **Accessibility (30+ regole)**: WCAG 2.1 AA compliance
2. **UX (40+ regole)**: Best practice consolidate
3. **Performance (20+ regole)**: Core Web Vitals optimization
4. **Visual Quality (30+ regole)**: Design polish e consistency

## Scoring
- **A (90-100%)**: Production-ready
- **B (80-89%)**: Deploy OK, fix minori consigliati
- **C (70-79%)**: Fix necessari prima di deploy
- **D/F (<70%)**: Rework obbligatorio

## Integration
Usa come ultimo step prima del deployment, dopo self-check CRO.
